## 1. 概述
- **v1.1 (2026-02-24)**: 侧重于用户界面(UI)的全面升级，对标 Google Gemini 的极简深色设计风格，并修复后端服务稳定性问题。

## 2. UI重构与体验优化详情

### 2.1 设计风格升级
- **Gemini Style**: 采用极简深色主题（背景色 `#0b0b0f`），移除冗余的装饰性元素（如光斑、玻璃拟态卡片）。
- **沉浸式阅读**: 详情页（博客、工具、教程）移除 `glass-card` 容器，内容直接在背景上呈现，减少视觉隔阂。
- **品牌一致性**: 
  - `Navbar` 在所有页面显示 "Lify" Logo。
  - Logo 位置调整为左对齐，增强品牌识别度。
- **列表页优化**:
  - 统一使用 `ContentCard` 组件，确保不同模块间视觉一致。
  - 移除冗余的 `h2` 标题，采用渐变文字样式的列表头部。

### 2.2 后端稳定性修复
- **错误处理增强**: 
  - 修复 `/tutorials` 接口 500 错误。
  - 在 `utils/errors.ts` 中补充 `UnauthorizedError` 和 `ForbiddenError` 导出，完善错误类型体系。
